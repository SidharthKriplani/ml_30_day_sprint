# ML 30 Day Sprint

A structured applied machine learning sprint focused on repeatedly taking datasets from raw form to model, evaluation, business decision, and production-shaped artifact.

## Structure

- Week 1: Core applied ML reps
- Week 2: Recommendations, experimentation, fraud, NLP, production handoff
- Week 3: Senior ML topics — forecasting, Optuna, drift, uplift, images
- Week 4: MLE topics — feature store, streaming inference, CI/CD, calibration, capstone

## Daily Output Standard

Each project folder will contain:

- README
- notebook or script
- metrics
- segment/error analysis
- business decision
- production-shaped artifact
